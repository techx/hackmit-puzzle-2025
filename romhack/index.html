Welcome to ROM Hack!
Author: kosinw
Try out my new Pokémon ROM hack. There might be something new in the Pokémon Center...Click around to explore!
To run the challenge, extract the tarfile in the PC and apply the romhack.ips patch to a Gameboy Advance ROM (you will need to find it yourself).

If the patch was applied correctly, the sha1 checksum should be:

6b7a9d4a1a870fb4dfd276c95a0418bde321163b romhack.gba

You will need a Gameboy Advance emulator to run the patched ROM. I recommend mGBA, as it is fast and accurate.

The flag format should match /hack\([A-Za-z0-9!]+\)/.

Replace "(" and ")" with "{" and "}" before submitting the flag.
romhack.tar.gz .............................................. 23M
├── romhack.ips ............................................. 8.3M
└── romhack.elf ............................................. 48M
